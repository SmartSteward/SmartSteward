# STM32相关控制代码说明

## 定时器相关说明

- 定时器二：定时触发电机PWN相关计算，并设置pwn值
- 定时器三：通过获取定时器计数器的值进而得到电机编码器的值
- 定时器五：通过获取定时器计数器的值进而得到电机编码器的值
- 定时器四：通过设置多个ccr通道进而控制多路电机占空比

## 串口输入与控制行为表

|十六进制字符 | 行为 |
| - | - |
|  41 |  前进 |
|  42 |  右前 |
|  43 |  向右 |
|  44 |  右后 |
|  45 |  后退 |
|  46 |  左后 |
|  47 |  向左 |
|  48 |  左前 |
|  5A |  停止 |
|  59 |  减速 |
|  58 |  加速 |
